Etapa 1 - Proposta do Projeto

Nome do sistema

Biblioteca Facil

Descricao do problema

O controle de uma biblioteca pequena pode ficar desorganizado quando e feito de forma manual. Em alguns casos, as informacoes sobre livros, usuarios, emprestimos e devolucoes ficam espalhadas em cadernos ou planilhas, o que dificulta a consulta e aumenta a chance de erro.

Com isso, pode ser dificil saber se um livro esta disponivel, quem esta com determinado livro emprestado ou quais emprestimos ainda precisam ser finalizados.

Objetivo do sistema

O objetivo do Biblioteca Facil e criar um sistema simples, feito em Java, para auxiliar no controle basico de uma biblioteca. O sistema deve permitir o cadastro de livros e usuarios, alem de registrar emprestimos e devolucoes.

O projeto tambem tem como objetivo aplicar os conceitos estudados em Programacao Orientada a Objetos, como classes, objetos, encapsulamento, heranca e polimorfismo.

Funcionalidades principais

  1. Cadastrar livros no sistema.
  2. Listar os livros cadastrados.
  3. Cadastrar usuarios da biblioteca.
  4. Listar usuarios cadastrados.
  5. Realizar emprestimo de livro para um usuario.
  6. Realizar devolucao de livro.
  7. Consultar emprestimos ativos.
